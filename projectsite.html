PiHole DNS Filter



Install Pi Imager from the pi-hole.net website. From there install the RPI OS Lite on the memory card, set a username and password along with other configuration such as WiFi and keyboard layout. From there you can SSH into the Pi via the Powershell or CMD terminal by navigating to the Pi's IP and using port 22. (Make sure that your router doesn't block this in ACLs)

Once connected to the Pi run the "sudo apt update && sudo apt upgrade -y" command to update the repositories on the device. Then run the install command for the PiHole "wget -O basic-install.sh https://install.pi-hole.net sudo bash basic-install.sh"

From here you can adjust the DNS settings and select a list of different known DNS servers that host adds and other unwanted redirects. I also recommend using the 9.9.9.9 cloudflare DNS server as a forward lookup DNS since it has DNSSEC which helps secure your traffic and lessens the risk of getting a browser hijack.

Finally you can assign your Pi's IP to the DNS server within your network and reconnect your devices to make this your new DNS server.

(Note: I have my Pi in a different VLAN than the rest of my traffic so I had to create rules within my router to allow port 53 with TCP/UDP to be communicated between for all other VLANs so that the devices could send DNS requests to the Pi.)
